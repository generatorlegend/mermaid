# Supported Diagrams in Mermaid

Mermaid is a powerful diagramming and charting tool that allows you to create various types of diagrams using text-based syntax. This guide provides an overview of all the supported diagram types in Mermaid, along with brief descriptions and links to more detailed documentation for each.

## Flowchart

Flowcharts are diagrams that represent workflows or processes. They are widely used in various fields to document, study, plan, and communicate complex processes in clear, easy-to-understand diagrams.

[Learn more about Flowcharts](./flowchart.md)

## Sequence Diagram

Sequence diagrams describe how entities interact with each other and in what order. They're excellent for documenting and understanding how different parts of a system work together.

[Learn more about Sequence Diagrams](./sequence-diagram.md)

## Class Diagram

Class diagrams are the main building block of object-oriented modeling. They show the classes in a system, attributes and operations of each class, and the relationships between classes.

[Learn more about Class Diagrams](./class-diagram.md)

## State Diagram

State diagrams, also known as state machine diagrams, are used to describe the behavior of a system. They illustrate the different states that an object can be in and how the object transitions between these states.

[Learn more about State Diagrams](./state-diagram.md)

## Entity Relationship Diagram (ERD)

Entity Relationship Diagrams (ERDs) are used to visualize the relationships between entities in a database. They're essential for designing and understanding database structures.

[Learn more about Entity Relationship Diagrams](./entity-relationship-diagram.md)

## User Journey Diagram

User Journey Diagrams help visualize the process a user goes through to accomplish a goal. They're particularly useful in user experience (UX) design and customer journey mapping.

[Learn more about User Journey Diagrams](./user-journey.md)

## Gantt Chart

Gantt charts are used in project management to illustrate a project schedule. They show the start and finish dates of different elements of a project, such as tasks or events.

[Learn more about Gantt Charts](./gantt.md)

## Pie Chart

Pie charts are circular statistical graphics which are divided into slices to illustrate numerical proportion. They're useful for showing the composition of a whole in terms of its parts.

[Learn more about Pie Charts](./pie-chart.md)

## Requirement Diagram

Requirement diagrams are used in systems engineering to provide a visualization of system requirements and their connections. They help in managing complex systems and ensuring all requirements are met.

[Learn more about Requirement Diagrams](./requirement-diagram.md)

## Gitgraph Diagram

Gitgraph diagrams are used to represent Git workflows. They're particularly useful for visualizing and documenting Git branching strategies and release processes.

[Learn more about Gitgraph Diagrams](./gitgraph.md)

## C4 Diagram

C4 diagrams are a set of hierarchical diagrams that are used to describe and communicate software architecture at different levels of abstraction. They include Context, Container, Component, and Code diagrams.

[Learn more about C4 Diagrams](./c4c.md)

## Mindmap

Mindmaps are diagrams used to visually organize information. They are often created around a single concept, drawn as an image in the center of a blank page, to which associated representations of ideas such as images, words and parts of words are added.

[Learn more about Mindmaps](./mindmap.md)

## Timeline Diagram

Timeline diagrams are used to display a list of events in chronological order. They're useful for project planning, historical overviews, and any other time-based visualizations.

[Learn more about Timeline Diagrams](./timeline.md)

## Quadrant Chart

Quadrant charts, also known as matrix diagrams, are used to show relationships between two variables. They're often used in business and strategy planning to categorize and prioritize information.

[Learn more about Quadrant Charts](./quadrant-chart.md)

## XY Chart

XY charts, also known as scatter plots, are used to display values for two variables for a set of data. They're useful for showing the relationship between two variables and identifying patterns in data.

[Learn more about XY Charts](./xy-chart.md)

By supporting this wide range of diagram types, Mermaid enables developers and teams to create diverse visualizations for various purposes, from software architecture and database design to project management and data analysis.
